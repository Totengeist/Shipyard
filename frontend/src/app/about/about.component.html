Welcome to the Shipyard!

Bringing you the best content for The Last Starship, whether you play on Steam, GOG, Humble Bundle, or SendOwl. Just like the game, we're currently in alpha and have a lot of exciting features planned.

Changes

This website may change frequently as we stability the look, feel, and feature set. You can view the changelog on GitHub.

Contributing

If you have experience with PHP/Slim and JavaScript/Angular and are interested in helping out, the project's code is hosted on GitHub. You can submit bug reports or fixes there.

If the project has provided you with some value, consider helping offset server costs by donating a dollar. The money will be used to support Shipyard and the TLS wiki.

Contact

If you need support or have questions, you can contact Totengeist on the TLS Discord server.
